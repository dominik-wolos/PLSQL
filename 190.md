# Zadanie 190
Wskazać trzy przedmioty, z których przeprowadzono najwięcej egzaminów.
W odpowiedzi umieścić nazwę przedmiotu oraz liczbę egzaminów.
W poprzednim miesiącu.

## Pytania tego typu odnoszą się do wartości.
1. Agregujemy egzaminy
2. Sortujemy
3. Iterujemy po wartościach
4. Prezentujemy przedmioty dla ktorych liczba egzaminow jest w top 3 znalezionych wynikow.

## Alternatywa

------- rowcount -------

Wybierz top 3 wartości
Znajdz przedmioty dla których szukana wielkość jest wśród top 3 wartości.

Zadanie
```sql
declare
	cursor c1 is select count(*) ExamNum from egzaminy group by id_przedmiot order by 1 desc ;
  cursor c2 (pExamNum number) is select nazwa_przedmiot from przedmioty p inner join egzaminy e
  																on p.id_przedmiot = e.id_przedmiot
                                  group by nazwa_przedmiot
                                  having count(*) = pExamNum ;
begin
	for vc1 in c1 loop
  	exit when c1%rowcount > 3 ;
  	--dbms_output.put_line(vc1.ExamNum) ; 
    for vc2 in c2(vc1.ExamNum) loop
    		dbms_output.put_line(vc2.nazwa_przedmiot || ' (' || vc1.ExamNum || ')') ; 
    end loop ;
	end loop ; 
end ; 
```
